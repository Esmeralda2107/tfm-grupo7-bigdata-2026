# 📎 Anexos del Proyecto

Este directorio reúne la **documentación complementaria** del proyecto, organizada en materiales de apoyo que ayudan a interpretar, contextualizar y comunicar distintos componentes metodológicos, analíticos y visuales del repositorio.

Los anexos se organizan por temática:

- **`clusterizacion/`**: contiene material de apoyo relacionado con los métodos de clusterización considerados en el proyecto y su contexto metodológico.
- **`modelo_datos/`**: reúne documentación visual y conceptual sobre la estructura lógica y relacional de los datos del proyecto.
- **`scoring/`**: incluye documentación metodológica del sistema de scoring multicriterio, sus dimensiones, variables y ponderaciones locales.
- **`escenarios/`**: contiene material explicativo sobre la construcción de escenarios de decisión y la lógica de ponderación entre dimensiones.
- **`streamlit/`**: documenta la aplicación interactiva desarrollada para la exploración visual de resultados.
- **`resultados/`**: reúne visualizaciones y piezas gráficas complementarias asociadas a los resultados finales del modelo.

En conjunto, esta carpeta actúa como un espacio de **apoyo documental**, orientado a complementar la información contenida en **`datos/`**, **`notebooks/`**, **`scripts/`**, **`sql/`**, **`resultados/`** y en la aplicación de **Streamlit**.
