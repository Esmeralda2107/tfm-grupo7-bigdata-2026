# 📎 Anexo de Scoring

Este directorio contiene material complementario de apoyo relacionado con la fase de **scoring multicriterio** del proyecto.

Su función principal es documentar, desde una perspectiva metodológica, la lógica de construcción del sistema de puntuación utilizado para priorizar las zonas de Manhattan según su compatibilidad con las necesidades estratégicas de DonTaquirou.

En esta carpeta se incluye el siguiente documento:

- **`metodologia_scoring_multicriterio.md`**: documento explicativo que describe la estructura general del scoring, la transformación de variables a una escala común, la lógica de sentido estratégico de las variables, la ponderación local por dimensión y la tabla resumen de variables, pesos y descripciones analíticas del modelo.

Este anexo complementa la documentación técnica desarrollada en **`notebooks/05_scoring/`** y **`resultados/05_scoring/`**, aportando un marco metodológico que facilita la comprensión de la fase de valoración multicriterio del modelo de localización.
