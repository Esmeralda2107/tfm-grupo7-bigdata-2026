# 📎 Anexo de Modelo de Datos

Este directorio contiene material complementario de apoyo relacionado con la **estructura lógica y relacional de los datos** del proyecto.

Su función principal es documentar, desde una perspectiva conceptual, la organización de las entidades que componen el sistema de información del TFM y las relaciones establecidas entre ellas para la implementación del modelo de base de datos.

En esta carpeta se incluye el siguiente documento:

- **`esquema_modelo_logico_tfm.pdf`**: representación visual del **modelo entidad–relación** del sistema de site selection, en la que la entidad **`ZONAS`** actúa como núcleo del modelo y se relaciona con las dimensiones de **censo**, **coste de alquiler**, **lugares de interés**, **movilidad**, **restaurantes** y **seguridad**.

Este anexo complementa la documentación técnica desarrollada en **`sql/`** y en **`datos/base_datos/`**, aportando una visión estructural del modelo lógico que facilita la comprensión de la arquitectura de datos del proyecto.
