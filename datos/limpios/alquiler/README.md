# 🗂️ Alquiler Limpio

Este directorio contiene la **base de alquiler comercial en su versión limpia**, utilizada en el proyecto como insumo para la dimensión de precio de alquiler.

A diferencia del resto de fuentes, esta información no procede de una única descarga cruda, sino de un proceso de **construcción propia** a partir de referencias documentales y fuentes de mercado. Por ello, su incorporación al pipeline comienza directamente en esta etapa, con una estructura ya tabular y preparada para su uso posterior.

La base resultante incorpora información organizada por **zona, códigos territoriales de referencia, áreas comerciales y precios estimados de alquiler**, constituyendo el insumo empleado en las fases posteriores de integración territorial y construcción del dataset maestro.
