# 🗂️ Alquiler Limpio

Este directorio contiene la **base de alquiler comercial en su versión limpia**, utilizada en el proyecto como insumo para la dimensión de precio de alquiler.

A diferencia del resto de fuentes, esta información no procede de una descarga cruda única, sino de un proceso de **construcción propia** a partir de referencias documentales y fuentes de mercado. Por ello, su incorporación  comienza directamente en esta etapa, con una estructura ya tabular y preparada para su uso posterior.

La base reúne información vinculada a **zonas, códigos territoriales de referencia, áreas comerciales y precios estimados de alquiler**, constituyendo el insumo utilizado posteriormente para la integración territorial y la consolidación del dataset maestro.
