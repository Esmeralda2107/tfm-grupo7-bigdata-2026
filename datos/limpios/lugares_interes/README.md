# 🗂️ Lugares de Interés Limpios

Este directorio contiene la **base de lugares de interés en su versión limpia**, resultado del procesamiento aplicado sobre la fuente original de equipamientos y puntos de interés urbanos utilizada en el proyecto.

En esta etapa, la información fue depurada y estructurada para conservar los atributos relevantes para el análisis territorial, incluyendo elementos relacionados con **nombre del lugar, categoría funcional, tipología del equipamiento y localización geográfica**.

Esta base constituye la versión tratada de la dimensión de lugares de interés y sirve como insumo para las etapas posteriores de integración espacial y consolidación del dataset maestro.
