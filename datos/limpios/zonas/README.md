# 🗂️ Zonas Limpias

Este directorio contiene la **base de zonas en su versión limpia**, resultado del procesamiento aplicado sobre la cartografía original de las unidades territoriales de Manhattan utilizada en el proyecto.

En esta etapa, el archivo GeoJSON fue filtrado, depurado y estructurado para conservar exclusivamente las geometrías y atributos territoriales relevantes para el análisis. El tratamiento incluyó la selección del ámbito espacial de Manhattan, la validación de identificadores y la organización de la capa geográfica que sirve como referencia común para el resto de fuentes.

La base resultante incorpora información relacionada con **identificadores territoriales, nombres de zona, atributos espaciales y geometrías poligonales**, constituyendo la versión tratada de la dimensión territorial empleada posteriormente en las fases de integración espacial y construcción del dataset maestro.
