# 🗂️ Seguridad Limpia

Este directorio contiene la **base de seguridad en su versión limpia**, resultado del procesamiento aplicado sobre la fuente original de incidentes y denuncias policiales utilizada en el proyecto.

En esta etapa, la información fue depurada, filtrada y transformada para conservar únicamente los registros y atributos relevantes para el análisis territorial. El tratamiento incluyó la selección espacial de Manhattan, la depuración de registros inconsistentes, la reclasificación de tipologías delictivas y la estructuración de variables útiles para su posterior agregación por zona.

La base resultante incorpora información relacionada con **tipo de delito, localización del incidente, características del hecho y coordenadas geográficas**, constituyendo la versión tratada de la dimensión de seguridad empleada posteriormente en las fases de integración espacial y construcción del dataset maestro.
