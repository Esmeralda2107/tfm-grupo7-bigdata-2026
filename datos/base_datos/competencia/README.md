# 🗂️ Competencia en Base de Datos

Este directorio contiene la **entidad de competencia** en su versión estructurada para base de datos, resultado de la integración territorial de la fuente gastronómica previamente depurada y clasificada.

En esta fase, los registros de establecimientos se organizaron de forma compatible con el modelo relacional del proyecto y se vincularon al sistema territorial de referencia, permitiendo su posterior agregación y explotación analítica.

La tabla resultante incorpora información relacionada con la **identificación del establecimiento, localización, tipología gastronómica, clasificación de competencia e identificadores territoriales**, constituyendo la dimensión de competencia dentro de la base de datos del proyecto.
