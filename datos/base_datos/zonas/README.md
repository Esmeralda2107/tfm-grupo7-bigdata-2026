# 🗂️ Zonas en Base de Datos

Este directorio contiene la **entidad territorial base** del proyecto en su versión estructurada para base de datos, resultado del tratamiento aplicado sobre la cartografía de Manhattan y de su adaptación al modelo relacional.

En esta fase, la información espacial se organizó como tabla de referencia principal, permitiendo la vinculación del resto de entidades mediante un identificador territorial común y consistente.

La tabla resultante incorpora información relacionada con **identificadores territoriales, nombres de zona y atributos espaciales de referencia**, constituyendo la entidad central del sistema de base de datos y el punto de enlace para el resto de dimensiones del proyecto.
