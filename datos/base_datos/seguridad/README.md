# 🗂️ Seguridad en Base de Datos

Este directorio contiene la **entidad de seguridad** en su versión estructurada para base de datos, resultado de la integración territorial de los registros delictivos previamente depurados y reclasificados.

En esta fase, la información se organizó de forma compatible con el modelo de datos del proyecto, permitiendo su uso posterior en análisis agregados por zona y en la construcción del dataset maestro.

La tabla resultante incorpora información relacionada con la **tipología del delito, localización del incidente, características básicas del hecho e identificadores territoriales**, constituyendo la dimensión de seguridad dentro de la base de datos del proyecto.
