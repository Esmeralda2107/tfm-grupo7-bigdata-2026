# 🗂️ Alquiler en Base de Datos

Este directorio contiene la **entidad de alquiler comercial** en su versión estructurada para base de datos. Reúne la información elaborada para esta dimensión y la adapta al modelo de datos del proyecto, de modo que pueda integrarse con el resto de entidades territoriales analizadas.

En esta fase, los datos se organizaron para garantizar su correspondencia con las zonas del área de estudio, incorporando identificadores territoriales de referencia que permiten su vinculación dentro del sistema relacional y su posterior uso en la construcción del dataset maestro.

La tabla resultante incluye información de **zona, identificadores territoriales y precio estimado de alquiler comercial**, constituyendo la versión estructurada y preparada para consulta de esta dimensión dentro de la base de datos del proyecto.
