# 🗂️ Movilidad en Base de Datos

Este directorio contiene la **entidad de movilidad** en su versión estructurada para base de datos, resultado de la integración territorial de la información de estaciones y métricas agregadas de flujo de pasajeros.

En esta etapa, la información fue organizada en una tabla compatible con el modelo relacional del proyecto, permitiendo su uso posterior en análisis comparativos por zona y en la construcción del dataset maestro.

La tabla resultante incorpora variables relacionadas con **estaciones, intensidad de flujo, temporalidad, métricas agregadas de movilidad e identificadores territoriales**, constituyendo la dimensión de movilidad dentro de la base de datos del proyecto.
