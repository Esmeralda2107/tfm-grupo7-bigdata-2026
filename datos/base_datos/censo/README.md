# 🗂️ Censo en Base de Datos

Este directorio contiene la **entidad censal** en su versión estructurada para base de datos, resultado de la consolidación e integración de los bloques sociodemográficos procesados en las etapas previas de limpieza.

En esta fase, la información censal se organiza en una tabla única, consistente y preparada para su incorporación al modelo relacional del proyecto, así como para su posterior uso en la construcción del dataset maestro.

La tabla resultante reúne variables vinculadas a **población, edad, origen hispano, tipo de hogar, empleo, ingresos, ocupación de vivienda y otros indicadores sociodemográficos relevantes**, constituyendo la dimensión censal consolidada dentro de la base de datos del proyecto.
