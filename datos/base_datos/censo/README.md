# 🗂️ Censo en Base de Datos

Este directorio contiene la **entidad censal** en su versión estructurada para base de datos, resultado de la consolidación e integración de los bloques sociodemográficos trabajados previamente en la etapa de limpieza.

En esta fase, la información censal fue organizada en una tabla única y consistente, preparada para su incorporación al modelo relacional del proyecto y para su posterior uso en la construcción del dataset maestro.

La tabla resultante reúne información vinculada a **población, edad, origen hispano, tipo de hogar, empleo, ingresos, ocupación de vivienda y otras variables sociodemográficas clave**, constituyendo la dimensión censal consolidada dentro de la base de datos.
