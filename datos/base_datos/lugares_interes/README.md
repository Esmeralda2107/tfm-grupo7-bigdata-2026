# 🗂️ Lugares de Interés en Base de Datos

Este directorio contiene la **entidad de lugares de interés** en su versión estructurada para base de datos, resultado de la integración territorial de la fuente de equipamientos y puntos de interés urbanos.

En esta etapa, la información fue organizada en una estructura compatible con el modelo de datos del proyecto, permitiendo su explotación posterior a nivel territorial y su incorporación al dataset maestro.

La tabla resultante incorpora información relacionada con **nombre del lugar, categoría funcional, tipología del equipamiento, localización e identificadores territoriales**, constituyendo la dimensión de lugares de interés dentro de la base de datos del proyecto.
