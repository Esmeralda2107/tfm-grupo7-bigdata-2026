# 🗂️ Zonas Crudas

Este directorio contiene la **fuente original de zonas** utilizada en el proyecto. Su contenido corresponde a un archivo **GeoJSON** con la cartografía base de las unidades territoriales de Manhattan empleadas como referencia espacial para la integración del resto de fuentes.

La fuente reúne información sobre **identificadores territoriales, nombres de zona, atributos espaciales y geometrías poligonales**, elementos fundamentales para la delimitación del área de estudio, la referencia cartográfica del proyecto y la asignación territorial de los datos.
