# 🗂️ Censo Crudo por Zona

Este directorio reúne los **31 archivos originales del Censo** correspondientes a las zonas de Manhattan utilizadas en el proyecto. Cada archivo representa una unidad territorial específica y conserva la estructura de origen con la que fue descargado, sin transformación analítica previa.

Estos archivos contienen información sociodemográfica de base, incluyendo variables relacionadas con **población, edad, origen hispano, estructura del hogar, empleo, ingresos, ocupación de vivienda y renta residencial**, entre otras dimensiones relevantes para el análisis posterior.
