# 🗂️ Competencia Cruda

Este directorio reúne la **fuente original de competencia gastronómica** utilizada en el proyecto. Su contenido corresponde a los registros crudos de inspecciones de restaurantes en Manhattan, empleados como base para la construcción de la dimensión de competencia.

La información contenida en esta fuente incluye variables relacionadas con la **identificación del establecimiento, localización, tipología gastronómica, resultados de inspección, infracciones sanitarias, calificaciones y coordenadas geográficas**, entre otras de interés para el análisis posterior.

Debido a las limitaciones de tamaño de archivo en GitHub, la fuente se encuentra almacenada en **formato comprimido**.
