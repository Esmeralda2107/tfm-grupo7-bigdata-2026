# 🗂️ Lugares de Interés Crudos

Este directorio contiene la **fuente original de lugares de interés** utilizada en el proyecto. Se trata del dataset crudo de equipamientos y puntos de interés urbanos de Manhattan, empleado como base para la construcción de la dimensión territorial de lugares de interés.

La fuente reúne información sobre **nombre del lugar, tipología de equipamiento, categorías funcionales, atributos administrativos y coordenadas geográficas**, entre otras variables relevantes para el análisis espacial del entorno.
