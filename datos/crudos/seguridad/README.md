# 🗂️ Seguridad Cruda

Este directorio contiene la **fuente original de seguridad** utilizada en el proyecto. Su contenido corresponde a los registros crudos de incidentes y denuncias policiales en Manhattan, empleados como base para la construcción de la dimensión de seguridad urbana.

La fuente reúne información sobre **tipología del delito, fecha y hora del incidente, localización, características del hecho y coordenadas geográficas**, entre otras variables relevantes para el análisis territorial posterior del riesgo y la seguridad.

Debido a las limitaciones de tamaño de archivo en GitHub, el archivo se encuentra almacenado en **formato comprimido**.
