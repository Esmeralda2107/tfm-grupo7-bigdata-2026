# 🗂️ Movilidad Cruda

Este directorio contiene la **fuente original de movilidad** utilizada en el proyecto. Su contenido corresponde a los registros crudos del sistema de transporte de Manhattan, empleados como base para la construcción de la dimensión de movilidad urbana.

La fuente reúne información sobre **validaciones o ridership, estaciones, horarios, fechas, flujos de pasajeros y coordenadas geográficas**, entre otras variables relevantes para el análisis de la intensidad de movilidad y la accesibilidad territorial.

Debido a las limitaciones de tamaño de archivo en GitHub, los archivos se encuentran almacenados en **formato comprimido**.
